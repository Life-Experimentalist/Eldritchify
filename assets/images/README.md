### Recommended Usage:

**For Background Image:**
Add to the CSS in `index.html`:
```css
body {
  background-image: url('assets/images/cthulhu-concept.jpg');
  background-size: cover;
  background-position: center;
  background-attachment: fixed;
  background-blend-mode: overlay;
}
```

**For README Banner:**
Update README.md header:
```markdown
![Eldritchify Banner](assets/images/cthulhu-concept.jpg)
```

### Image Guidelines:
- **Format**: JPG, PNG, WebP
- **Optimization**: Compress large images for web performance
- **Attribution**: Credit artists when using their work
- **Theme**: Keep images aligned with Lovecraftian/cosmic horror aesthetic
